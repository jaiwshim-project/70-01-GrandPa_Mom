## STEP 2: Viewer가 데이터 로드

```
╔═══════════════════════════════════════════════════════════════════════════════════════════════════╗
║                                                                                                   ║
║   STEP 2: JSON Method - Viewer가 데이터를 로드하는 과정                                           ║
║                                                                                                   ║
║   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐ ║
║   │                                                                                             │ ║
║   │   🌐 viewer_json.html                                                                       │ ║
║   │                                                                                             │ ║
║   │   ┌─────────────────────────────────────────────────────────────────────────────────────┐   │ ║
║   │   │                                                                                     │   │ ║
║   │   │   // 1. 로컬 또는 GitHub Pages에서 실행                                             │   │ ║
║   │   │   // 상대 경로로 JSON 파일 접근                                                     │   │ ║
║   │   │                                                                                     │   │ ║
║   │   │   // 2. index.json 로드 (Task ID 목록)                                              │   │ ║
║   │   │   const indexRes = await fetch('../method/json/data/index.json');                   │   │ ║
║   │   │   const { task_ids } = await indexRes.json();                                       │   │ ║
║   │   │                                                                                     │   │ ║
║   │   │   // 3. 개별 Task JSON 병렬 로드                                                    │   │ ║
║   │   │   const tasks = await Promise.all(                                                  │   │ ║
║   │   │       task_ids.map(id =>                                                            │   │ ║
║   │   │           fetch(`../method/json/data/grid_records/${id}.json`)                      │   │ ║
║   │   │               .then(r => r.json())                                                  │   │ ║
║   │   │       )                                                                             │   │ ║
║   │   │   );                                                                                │   │ ║
║   │   │                                                                                     │   │ ║
║   │   │   // 4. Stage Gate JSON 로드                                                        │   │ ║
║   │   │   const gates = await Promise.all(                                                  │   │ ║
║   │   │       [1,2,3,4,5].map(s =>                                                          │   │ ║
║   │   │           fetch(`../method/json/data/stage_gate_records/S${s}_gate.json`)           │   │ ║
║   │   │               .then(r => r.json())                                                  │   │ ║
║   │   │       )                                                                             │   │ ║
║   │   │   );                                                                                │   │ ║
```

```bash

npx serve
```

```bash
gh auth login

git commit -m "Initial commit"
gh repo create {프로젝트명} --public --source=. --push
```
